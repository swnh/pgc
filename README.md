# Point Cloud Compression (PGC)

This project implements point cloud compression using predictive geometry encoding based on MPEG PCC TMC13.

## Project Structure

- `source/` - Source code including custom implementations and TMC13 integration
- `scripts/` - Build and test scripts
- `datasets/` - Point cloud datasets (not included in repository)
- `experiments/` - Experimental results and configurations
- `docs/` - Documentation
